# Telegram Channel Poster (Test)

This project sends a single test message to your Telegram public channel.

## Environment Variables
- BOT_TOKEN: your bot token from BotFather
- CHAT_ID: your public channel username, e.g. @MyChannel

## Run locally
pip install -r requirements.txt
export BOT_TOKEN="..."
export CHAT_ID="@MyChannel"
bash start.sh
